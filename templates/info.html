About Phishing

Phishing is a type of cyber attack where attackers impersonate legitimate organizations or individuals to trick victims into providing sensitive information, such as passwords, credit card numbers, or other personal data. Phishing attacks can occur through email, social media, text messages, or even phone calls.

Common signs of phishing include:

  • Unexpected or unsolicited messages
  • Requests for sensitive information
  • Urgent or threatening language
  • Suspicious links or attachments
  • Spelling and grammar errors

To protect yourself from phishing attacks, always verify the sender's identity, avoid clicking on suspicious links, and never provide sensitive information in response to unsolicited requests.

Phishing Methods:


Email Phishing:

Using organization’s actual name but using a fake address (e.g., paypal@domainregistrar.com)

Spear Phishing:

Targets specific individuals by using their personal information

Smishing & Vishing:

Smishing uses text messages, while Vishing uses phone calls to get a person to click a link or share personal information

Anger Phishing:

Uses social media to get information or trick someone into downloading malware or possibly geotagging


About Malware:

Malware is malicious software designed to harm or exploit any programmable device, network, or user. Malware can be used to steal sensitive information, disrupt operations, or gain unauthorized access to systems. It can be delivered through various means, including email attachments, infected websites, or software vulnerabilities.

Malware Types:


Ransomware:

Designed to deny a user to a system, network, data, or hardware until ransom is paid

Trojans:

Disguises itself as a program that seems safe, and run regardless of an application running in the background

Viruses:

A self-replicating program that inserts itself into an application to execute a specific task while app is running

Worms:

A malicious program that replicates itself to automatically spreading through a network and consume large volumes or memory and run on its own
